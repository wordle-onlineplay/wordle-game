## 🕹️ How to Play Wordle

1. **Type a 5-Letter Word**: Make your first guess.
2. **Use Feedback**: Green = correct spot, yellow = correct letter/wrong spot, gray = not in word.
3. **Keep Guessing**: Narrow it down in 6 tries or fewer.
